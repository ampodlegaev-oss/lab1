# Циклический сдвиг по прямоугольным рамкам

Веб-приложение для обработки изображений, реализующее циклический сдвиг пикселей по вложенным прямоугольным рамкам.

## Описание

Приложение позволяет загрузить изображение и выполнить циклический сдвиг пикселей вдоль периметра каждой вложенной прямоугольной рамки. Для каждой рамки пиксели перемещаются на заданное пользователем число позиций, при этом пиксели, выходящие за границы рамки, переносятся на противоположную сторону. Центральный пиксель изображений с нечётными размерами остаётся неизменным.

## Возможности

- Загрузка изображений в форматах PNG, JPEG, BMP, GIF
- Настройка величины сдвига
- Отображение исходного и обработанного изображений
- Визуализация распределения цветовых каналов (R, G, B)
- Защита от автоматических атак с использованием Google reCAPTCHA v2

## Используемые технологии

- Python 3.13
- Flask
- NumPy
- Pillow
- Matplotlib
- Gunicorn
- GitHub Actions
